## Abstract

This thesis presents VibeOS, an open-source, metadata-driven platform that compiles natural language intent into fully functional business applications. By leveraging Large Language Models (LLMs) to generate structured JSON schemas — termed "Vibe-JSON" — the system dynamically produces database structures, API endpoints, and user interfaces without manual coding. The platform is built on Next.js 15, PostgreSQL with JSONB storage, and the Vercel AI SDK, following a modular monolith architecture. This work demonstrates that the "Vibe Coding" paradigm can reduce application development time by an order of magnitude compared to traditional low-code platforms such as Salesforce.

**Keywords:** metadata-driven development, server-driven UI, natural language processing, large language models, low-code platforms, vibe coding
