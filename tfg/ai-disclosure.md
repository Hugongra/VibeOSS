## General Statement

AI tools were used in this project in two distinct capacities:

1. **As the core technology being researched** — The VibeOS platform integrates Large Language Models (specifically OpenAI GPT-4o via the Vercel AI SDK) to generate metadata schemas from natural language. This is the primary subject of the thesis.

2. **As a development assistant** — AI coding assistants were used to accelerate development, generate boilerplate, and refine code quality. All AI-generated code was reviewed, tested, and modified by the author.
